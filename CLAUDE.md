# Build commands

- `npm run build` ‚Äî build all platforms (Chrome, Firefox, Safari)
- `npm run build:firefox` ‚Äî build Firefox only (output: `dist_firefox/`)
- `npm run build:safari` ‚Äî build Safari only (output: `dist_safari/`)
- Xcode (macOS): `cd "xcode/Obsidian Web Clipper" && xcodebuild -scheme "Obsidian Web Clipper (macOS)" -configuration Debug build`

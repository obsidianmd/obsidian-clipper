AppFlowy Web Clipper is a free browser extension that lets you highlight pages and save web content to your AppFlowy workspace.

> This is an unofficial fork of [Obsidian Web Clipper](https://github.com/obsidianmd/obsidian-clipper), adapted for AppFlowy.

## Install Web Clipper

The extension is not yet published to browser stores. Install it locally from the [GitHub repository](https://github.com/alexrosepizant/clipper-for-appflowy) by following the developer instructions in the README.

## Privacy

AppFlowy Web Clipper saves content directly to your AppFlowy workspace. Your data is not collected by this extension. The code is [open source](https://github.com/alexrosepizant/clipper-for-appflowy) and auditable.
